Umbrella Calculator README

🛡️ Umbrella OS Tactical Calculator

A high-fidelity, web-based calculator themed after the Umbrella Corporation from the Resident Evil universe. This project features a "Biohazard" aesthetic with a terminal-inspired UI, industrial color palettes, and a layout designed to mimic high-end laboratory hardware.

Responsive Logic: Clean JavaScript implementation for arithmetic operations.

🛠️ Built With

HTML5: Semantic structure for the calculator chassis.
CSS3: Advanced Grid and Flexbox layouts for the industrial button array and positioning.
JavaScript (ES6): Logic engine for real-time calculation.

🚀 How to Run

Clone this repository or download the files.
Ensure your folder structure looks like this:
index.html
stylesheet.css
script.js
media/Umbrella_Corporation_Logo.pn
